# &AI MATCH

> PayPayマネーライトを現金・USDTに換金できる唯一のサービス

## 概要
PayPay残高（マネーライト）の換金ニーズに特化したマッチングプラットフォーム。
売り手と買い手をAIが最適マッチングし、安全なP2P取引を実現する。
手数料モデルで収益化、PayPayエコシステムの盲点を突く独自サービス。

## 主な機能
- PayPayマネーライト ↔ 現金/USDT換金マッチング
- AIによる最適価格・相手マッチング
- リアルタイム取引フロー（受取確認・エスクロー）
- 不正検知・本人確認（KYC）
- 取引履歴・レーティングシステム

## 技術スタック
- Next.js 15 / TypeScript
- Supabase（DB + Auth）
- PayPay API連携
- Stripe（手数料決済）
